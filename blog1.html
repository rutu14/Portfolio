Box Model

A box model helps in defining the css properties through it, from creation of layouts to aligning the items present. The CSS box model in its entirety applies to block boxes. Inline boxes only use a subset of the box model's behaviour. The model specifies how the various components of a box ,such as margin, border, padding, and content, interact to form a visible box on a page.

Parts of box model

Content box: The area in which your content is displayed, can be sized using characteristics like as width and height.

Padding box: The padding box is white space that lies around the content; its size can be modified using padding settings.

Border box: The border box surrounds the content and padding. The border parameters can modify its size and design.

Margin box: The margin box is the outermost layer, enveloping the content, padding, and border as whitespace between this box and other elements. Its size may be adjusted by adjusting the margin settings.



To view the box model on browser for an element, check the Computed tab in DevTools.
